# AAS Metamodel GraphQL Schema

This folder contains GraphQL schema for the AAS metamodel. It will try to remain as neutral and as indepentend as possible. Neo4j specific schema, or Submodel Template specific types will be hosted in other folders. 

Further Mutation, Query, and Subscription endpoint will be added.

## Contribution & Documentaitons

[decisions.md](decisions.md) holds informal documentation and contains questions/decisions in an informal way. If you are interested in making a contribution, feel free to contact us or raise an issue if you spot any problems.
